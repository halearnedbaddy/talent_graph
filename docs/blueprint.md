# **App Name**: Verve & Vigor - Talent Graph Landing Page

## Core Features:

- Hero Section Display: Showcase a headline and sub-headline that clearly articulates the platform's value proposition for athletes and scouts, with 'Create Account' and 'Sign In' call-to-action buttons.
- Problem Statement Display: Clearly outline the current problems in sports talent discovery, focusing on the lack of verifiable data and the abundance of noise.
- Solution Presentation: Present the Talent Graph as a professional identity layer, highlighting verified metrics, structured profiles, and long-term tracking.
- Step-by-Step Guide: Outline a linear 'How It Works' guide that demonstrates how users can create accounts, build profiles, track performance, and get discovered.
- User Persona Sections: Show dedicated sections for athletes and scouts, emphasizing benefits like building credibility, reducing noise, and improving talent discovery.
- Philosophy and Trust Section: Communicate a long-term philosophy focused on discipline and consistency, creating a sense of trust and stability.
- Account Creation Route: The Create Account and Sign In buttons in the app will route to /signup and /login to ensure proper function of user creation and authentication.

## Style Guidelines:

- Primary color: Bright Beige White (#F5F5DC) to create a clean and trustworthy feel.
- Secondary color: A Good Shade of Grey (#808080) to provide a neutral and professional balance.
- Accent color: Black (#000000) to highlight key actions and information, creating contrast against the lighter background.
- Body and headline font: 'Inter' sans-serif for an institutional, modern look.
- Mobile-first, responsive design to ensure credibility across devices.
- Simple, professional icons for clear navigation and information hierarchy.
